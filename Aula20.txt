Dicas para caso o GitHub apresente problemas:
1 - Forçar a atualização de remoto (repositório do GitHub) = git push --force
2 - Forçar atualização local (onde estou trabalhando) = git pull -- rebase

Rebase = rebasear (utiliza o meu branch local)

Quando iniciar lembrar de criar um arquivo chamado reset.css e colocá-lo acima do style.css para que a página não contenha dados prontos e possa ser estilizado por você.

Para ter uma noção de como funcionaa estilização:

            ___________________________________
    margin  |  Padding (preenchimento interno) |
    externo |       __________________         |   
            |       |Elemento do html|         |
            |       ------------------         |
            |__________________________________|

Nos códigos de margin ou padding podemos utilizar apenas uma infomração com "Xrem" valores: top (cima) right (direita) bottom (baixa) left (esquerda) / Quando a definição for igual para todos basta digitar apenas uma vez.

Em caso de possível perda no arquivo temos 2 opções:

1ª utilizar o rebuild para recarregar as informações ou o full rebuild para reiniciar todo o codespace.

2ª opção: apagar o codespace e criar outro.
